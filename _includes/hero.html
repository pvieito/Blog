OverPicture

for Safari macOS iOS

OverPicture is an Extension for Safari that allows you to play any web video in Picture-In-Picture mode.

MakePass

Barcode to Wallet macOS iOS

MakePass allows you to create Apple Wallet passes quickly from any supported barcode. MakePass supports scanning and detecting barcodes from images, PDFs and directly from the camera, customizing the pass and storing it on Wallet so it will be synced across all your devices.

MakePDF

Document Merger macOS

MakePDF allows you to merge multiple files in a unique PDF really quickly by simply dropping them in the app or using the Extension from the Finder.

AirWeight

for Bluetooth Scales macOS iOS

AirWeight connects to your SANITAS SBF70, Xiaomi Mi Scale and Xiaomi Mi Scale 2 Bluetooth LE scale and reads your weight in realtime, calculates your Body Mass Index and saves it to HealthKit.

InstaLogin

for Safari macOS

InstaLogin is a Safari App Extension that allows you to automatically login in websites instantly with Safari AutoFill!

DepthViewer

for Portrait Mode macOS

DepthViewer can load any Portrait Mode photo with Depth Maps taken with iPhone X and iPhone 7 Plus and show a 3D model of the image.

InstaRate

Widget for iTunes macOS

This app allows you to quickly rate your iTunes song both from the app and from its Notification Center widget. You can even do it from the Touch Bar!

OverVoice

Message Transcriber iOS

OverVoice allows you to transcribe any voice message or audio file using its Extension. It can convert voice messages to text in all languages supported by iOS dictation service.

SourceCounter

Programming Languages Analyzer macOS

SourceCounter analyzes the source code files in a folder, classifies them by their language and shows you a report with the code lines, number of files and the total size per language in a data table and an interactive plot.
